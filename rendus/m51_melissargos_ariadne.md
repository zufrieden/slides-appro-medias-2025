## Mon sujet d'étude est ...

Pour ce projet, mon objectif est d'examiner et d'évaluer CURSOR, un outil d'intelligence artificielle conçu pour le développement, afin de mieux comprendre comment il peut être intégré dans un workflow de développement logiciel et faciliter la gestion de projets complexes.

CURSOR est un IDE alimenté par l'IA qui permet une meilleure compréhension, exploration et amélioration de code source volumineux. Il propose des suggestions intelligentes pour expliquer le code, le refactoriser et optimiser la navigation dans les bases de code.

Dans le cadre de mon Travail de Bachelor, qui porte sur l’amélioration de l’interface et de la visualisation des données de la Medical Informatics Platform du CHUV, je vais approfondir l’utilisation de CURSOR pour :

- Comprendre et naviguer plus efficacement dans l’architecture front-end existante.
- Identifier les parties critiques du code nécessitant des corrections ou des améliorations UX/UI.

### Contexte

J’ai choisi ce sujet car mon Travail de Bachelor portera sur l’amélioration de l’interface utilisateur de la Medical Informatics Platform (MIP). Cette plateforme possède une base de code front-end complexe. Le front-end est en pleine refonte, il est donc important de pouvoir analyser rapidement l’état de la version actuelle afin de proposer des améliorations pertinentes et durables. Avec CURSOR, je devrai naviguer efficacement dans cette architecture pour identifier des améliorations et corriger des bugs.

Pourquoi c’est utile pour mon TB ?

- Faciliter l’exploration du code existant et identifier les zones critiques à optimiser.
- Accélérer la correction de bugs et l’amélioration de l’expérience utilisateur.
- Mieux structurer les tâches à effectuer en fonction des insights fournis par CURSOR.
- Documenter et expliquer clairement les modifications apportées.
  
Pourquoi c’est intéressant pour mon futur professionnel ?

- Développer des compétences avancées en débogage et refactoring de code.
- Apprendre à travailler avec des outils d'IA utilisés en développement logiciel.
- Renforcer ma capacité à analyser et structurer du code dans un environnement réel.
